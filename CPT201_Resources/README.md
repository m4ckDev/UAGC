# CPT201 Quick Reference Pack (Good-To-Go)

This folder is a fast reference kit for CPT201 Automation & Scripting for Cybersecurity.

## What's inside
### CheatSheets
- PowerShell and Python quick references, one-liners, and patterns used in scripting + security automation.

### QuickScripts
- Small, reusable scripts for day-to-day automation: logs, processes, networking checks, file ops, CSV work, hashing, and reporting.

### Templates
- Reusable templates for reports, documentation, and script headers.

### Samples
- Sample outputs and example data used for testing.

## How to use this pack
1. Copy a script from `/QuickScripts`
2. Run it in a safe environment
3. Modify the variables at the top
4. Save outputs to `/Samples/Output`

> These resources are designed for academic and defensive use: monitoring, reporting, automation, and basic troubleshooting.
